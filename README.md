# Video Production OS

Tools and playbooks for turning a messy client brief into a cinematic results film,
using Canva Pro + AI generation, no traditional editing software.

Built live while producing a real 60 second corporate performance film for a liquor
portfolio seminar (client details removed, method kept).

## Showcase
- Final video and designs: https://canva.link/aidesigns
- Canva template: https://canva.link/aoq6brm740mk1kw
