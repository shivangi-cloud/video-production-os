When stuck, read this first ▼
Overlay won't land / replaces the video: click the page first, then CLICK the file's thumbnail in Uploads. Never drag files onto a page that has a video.
Overlay slides across pages: it was dropped on the timeline. Delete it, then add it by clicking, it must live on the page, only music lives on the timeline.
Durations misbehave: always TYPE them via the clock icon. Never drag page edges, never press Sync now.
Trimming a clip: about 1 second, starting just before the action peaks, so the peak happens during the page.
Music feels off: don't sync 37 pages. Drag the whole track until its big drop lands on the hero number page, then check only the five number pages.
Need slow-then-fast drama: same clip on two pages, first half at 0.5x speed with text, second half at 2x with no text.
